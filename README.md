# Tandemloop Test - C++

## Language
C++

## Description
This repository contains solutions to the Tandemloop first screening programming test.  
There are 4 problems, each implemented in its own C++ file:

- `Program-1.cpp` → Calculator (Addition, Subtraction, Multiplication, Division)
- `Program-2.cpp` → Odd number series up to input `a`
- `Program-3.cpp` → Odd number series in alternating pattern
- `Program-4.cpp` → Count multiples in a list of numbers

## How to Run
1. Open PowerShell or terminal in this folder.
2. Compile a program using `g++`:

```powershell
g++ Program-1.cpp -o program1
